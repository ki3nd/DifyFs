# Privacy Policy

## Data Collection

DifyFS does not collect, store, or transmit any user data to third parties.

## Data Access

DifyFS connects exclusively to the Dify Service API endpoint you configure. All requests are made directly from your environment to your Dify instance. No data passes through any external server operated by the plugin author.

The following credentials are used solely to authenticate with your Dify instance:

- **Service API Endpoint** — the base URL of your Dify deployment
- **Dataset API Key** — a scoped API key used to read knowledge base documents

## Data Usage

DifyFS is a read-only plugin. It retrieves document content, metadata, and search results from your Dify knowledge base in order to respond to tool calls. No document content is stored, logged, or forwarded outside your Dify instance.

## Third-Party Services

DifyFS has no dependency on any third-party service beyond your configured Dify instance.

## Contact

For questions or concerns, contact the author at the repository linked in the plugin manifest.
